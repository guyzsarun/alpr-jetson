# ALPR-IoT

ALPR using Nvidia Jetson Nano and Raspberry Pi for (2147336) Internet of Things

## Components
---
- Nvidia Jetson Nano
- STM32F411RE
- Raspberry Pi4 x 2
- Azure VM
- Nvidia CUDA GPU (Tensorflow Serving)

## Models
---
- TensorRT SSD MobilenetV2
- OpenALPR
- TessaractOCR
- License Plate Detector
